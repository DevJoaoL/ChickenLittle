# 🐔 ChickenLittle

Aplicação interativa desenvolvida com foco em lógica de programação e controle de fluxo, com objetivo de simular comportamento e interação do usuário.

---

## 🚀 Tecnologias utilizadas

- C#
- HTML5
- CSS3

---

## ⚙️ Funcionalidades

- 🎮 Interação com o usuário
- 🔄 Execução de lógica e regras da aplicação
- 🧠 Uso de estruturas condicionais e loops
- 📊 Controle de fluxo do sistema

---

## 🧠 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Lógica de programação
- Organização de código
- Estruturação de aplicações completas
- Desenvolvimento de aplicações interativas
